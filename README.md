# Pizza Makers Neovim Config

## Keymapp

```lua
<leader>b  -- escape to netrw
<leader>t  -- toggle terminal
C-p        -- fuzzy find files
<leader>fg -- grep search
<Shift>k   -- show documentation
<leader>n  -- show file tree
```
